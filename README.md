# FCPD Crime Data Analysis (Python)

### Description
This project processes and analyzes real-world crime data from the Fairfax County Police Department using Python. It demonstrates object-oriented programming and data analysis techniques.

### Technologies Used
- Python
- CSV data processing

### Features
- Custom class extending Python list to store crime data
- Loads and parses CSV crime reports
- Filters data by zip code, type, and date
- Generates crime statistics by zip code
- Counts crime occurrences by type

### How to Run
1. Make sure all files are in the same folder:
   - IT209_Adel_McCray_A7.py
   - crime.csv
2. Run the script:
   python IT209_Adel_McCray_A7.py

### Project Files
- IT209_Adel_McCray_A7.py – Main Python script
- crime.csv – Dataset used for analysis
